Simple Joomla Backup - Backup de Portais Joomla

Bom para instalar seu portal em outro computador ou em um servidor web. Também é útil para gerar backup do seu portal.

Proporciona rapidez, agilidade e conforto consequente nas operações de backup e migração de portais Joomla.
Ele é rápido, pois escolhi uma classe interna do próprio PHP para compactar (ZipArchive).

Usando o Simple Joomla Backup:

- Instale o componente com_simplejoomlabackup
- Clique em Componentes - simplejoomlabackup
- Clique no botão Start
- Aguarde e faça o download
- Então Instale em qualquer computador com suporte a Joomla ou no seu próprio computador com outro nome.

Agora para restaurar basta:

- Criar o banco e importar o .sql para ele

- Descompactar o arquivo .zip criando uma pasta no documentRoot

- Editar o configuration.php e ajustar os dados do banco, $log_path e $tmp_path;

Agora pode chamar no navegador.


Como usar o Instalador do Joomla para restaurar seu backup

- Descompacte o arquivo .zip gerado pelo Simple Joomla Backup

- Faça o download do Joomla 2.5 da mesma versão do seu site
- Descompacte o Joomla
- Acesse a pasta descompactada e copie a sub-pasta "installation" para o raiz da pasta do backup

- Renomeie o arquivo .sql do backup para joomla.sql
- Copie joomla.sql sobrescrevendo o existente para a pasta installation/sql/mysql

- Delete ou renomeie o arquivo configuration.php da pasta do backup
- Compacte novamente a pasta do backup e agora poderá restaurar seu backup usando o instalador do Joomla


Em caso de Falha no Backup

Se por algum motivo o Simple Joomla Backup falhar na criação do backup, as duas melhores alternativas que conheço são:<br>
- Akeeba Backup - https://www.akeebabackup.com/download/akeeba-backup.html
  A configuração necessária é:
    Componentes - Akeeba Backup - Configuration
    Archiver engine - ZIP Format
- A outra alternativa é um backup manual, que não dá muito trabalho:
    Acesse o gerenciador de arquivos do cpanel
    Selecione a pasta do seu site
    Clique em Compress
    Faça o download do arquivo gerado
    Acesse o phpMyAdmin
    Selecione o banco do seu site
    Exporte o banco no formato default (.sql) e faça o download
- Restaurando o backup manual
    Idealmente um backup é restaurado em um site novo
    Apenas acesse o cpanel e descompacte o zip dos arquivos numa nova pasta
    Crie um novo banco e acesse o phpmyadmin para importar o script que gerou inicialmente
    Faça os devidos ajustes no configuration.php, se necessário

Erros

Caso apareça este erro ou algo parecido:
Fatal error: Out of memory (allocated 273154048) (tried to allocate 200046879 bytes)
Precisará aumentar o memory_limit do seu PHP. Geralmente precisa pedir suporte ao pessoal do servidor.
